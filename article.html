Compagnon de bureau qui Fait ses Crottes, Te Dit 'Je t'aime' et Répond à Mes Questions : Voici Comment il est né

Salut les renards ! Bienvenue sur the-fox.tech pour découvrir Minou, mon dernier projet complètement fou : un desktop pet intelligent qui vit sur votre bureau Windows/Mac/Linux.

Imaginez un Tamagotchi moderne qui se balade sur votre écran, fait littéralement ses besoins, vous dit "je t'aime" au hasard, surveille votre RAM, et répond intelligemment à vos questions grâce à l'IA Gemini. Inutile ? Peut-être. Adorable et techniquement intéressant ? Absolument.

On va décortiquer ensemble comment j'ai codé ça en PyQt5, avec des animations 2D fluides, une architecture modulaire, et même un système de... crottes réalistes. Oui, vous avez bien lu.

Le Concept : Pourquoi Créer un Pet Desktop ?

Franchement, qui n'a jamais eu envie d'un petit compagnon virtuel sur son bureau ? Les années 90-2000 avaient leurs Tamagotchis et leurs eSheep qui couraient partout sur Windows. Moi, j'ai voulu mixer cette nostalgie avec la tech moderne.

Ce que fait Minou :

  • Se balade : Animations fluides, rebonds sur les bords
  • Interagit : Répond à vos clics, se laisse caresser (avec modération)
  • Communique : Chat intelligent avec IA Gemini intégrée
  • Assiste : Notes, rappels, surveillance système
  • Fait ses besoins : Parce que c'est drôle et ça ajoute du réalisme

La Stack Technique : PyQt5 et Architecture

Pour ce projet, j'ai opté pour une approche desktop native avec PyQt5. Pourquoi pas Electron ou du web ? Simple : les performances, la consommation mémoire, et le contrôle total sur l'interface.

Le Cœur du Projet : Animations 2D Fluides

L'âme de Minou, ce sont ses animations. J'ai voulu quelque chose de fluide, avec une vraie personnalité. Chaque action a sa séquence de sprites, et le système gère les transitions intelligemment.

ANIMATION_FRAMES = {

def _set_animation(self, new_animation_name):
    """Change l'animation courante avec gestion des transitions"""
    if self.is_playing_one_shot_animation and new_animation_name != self.current_animation:
        return  # Empêche l'interruption d'animations critiques
        
    if new_animation_name not in self.sprites or not self.sprites[new_animation_name]:
        # Fallback sur Idle si l'animation n'existe pas
        new_animation_name = 'Idle'
        
    if self.current_animation != new_animation_name:
        self.current_animation = new_animation_name
        self.current_frame_index = 0
        self._update_cat_pixmap()

Le Truc Malin :

J'utilise un QTimer à 100ms pour changer les frames, et un autre à 16ms (~60 FPS) pour la physique. Ça donne un rendu ultra fluide même avec des sprites pixelisés.

L'Intelligence : Gemini AI et Système de Chat

Le vrai plus de Minou, c'est son cerveau. Double-cliquez dessus, et vous ouvrez une interface de chat minimaliste qui cache une vraie intelligence artificielle. Mais j'ai pensé à ceux qui n'ont pas d'API : il fonctionne aussi hors-ligne.

Gestion dual mode :

def process_message(self, message):
    """Traite les messages avec fallback intelligent"""
    
    # Commandes locales prioritaires
    if message.lower() in ["reste tranquille", "stop", "pause"]:
        return self.handle_quiet_mode()
    
    if "note que" in message.lower():
        return self.handle_note_taking(message)
    
    # IA Gemini si configurée
    if self.ai_enabled and self.gemini_client:
        try:
            return self.gemini_client.generate_response(message)
        except Exception as e:
            return self.fallback_response(message)
    
    # Fallback local
    return self.generate_local_response(message)

def _spawn_random_poop(self): 
    """Spawn automatique de crottes près de l'animal"""
    if not self.poop_sprites:
        return

    # Position relative à l'animal
    pet_x = int(self._current_x)
    pet_y = int(self._current_y)
    spawn_x = pet_x + CAT_WIDTH // 2 - POOP_SIZE // 2
    spawn_y = pet_y + CAT_HEIGHT - POOP_SIZE // 2 

    # Sprite aléatoire
    poop_path = random.choice(self.poop_sprites)
    poop_item = PoopItem(image_path=poop_path, initial_pos=QPoint(spawn_x, spawn_y))
    
    # Nettoyage au clic
    poop_item.poop_removed.connect(self._on_poop_removed)

def parse_reminder_time(self, message):
    """Parse intelligemment les indications de temps"""
    import re
    from datetime import datetime, timedelta
    
    # "dans X minutes"
    minutes_match = re.search(r'dans (\d+) minutes?', message.lower())
    if minutes_match:
        minutes = int(minutes_match.group(1))
        return datetime.now() + timedelta(minutes=minutes)
    
    # "dans X heures"
    hours_match = re.search(r'dans (\d+) heures?', message.lower())
    if hours_match:
        hours = int(hours_match.group(1))
        return datetime.now() + timedelta(hours=hours)
    
    # "à 15h30"
    time_match = re.search(r'à (\d{1,2})h(\d{2})?', message.lower())
    if time_match:
        hour = int(time_match.group(1))
        minute = int(time_match.group(2) or 0)
        # Logique pour aujourd'hui ou demain selon l'heure actuelle
        return self.calculate_reminder_datetime(hour, minute)

Personnalisation : Donnez-lui un Nom, Changez Tout

La beauté de Minou, c'est qu'il s'adapte à vous. Vous pouvez personnaliser absolument tout : son nom, le vôtre, sa vitesse, sa personnalité, et même désactiver les crottes si ça vous dérange.

Configuration Identity

  • Nom de l'animal (défaut: "Minou")
  • Votre nom (défaut: "theTigerFox")
  • Type : Chat ou Chien
  • Personnalité : Joueur, Calme, Énergique

Configuration Comportement

  • Vitesse de mouvement (1-10)
  • Fréquence des messages d'amour
  • Mode tranquille auto
  • Système de crottes activé/désactivé

Interface de Configuration :

Accessible via menu système → Paramètres. Sauvegarde automatique en JSON, interface moderne avec thème sombre, et application des changements en temps réel sans redémarrage.

Tester Minou : Installation et Première Configuration

Bon, assez de blabla. Si vous voulez tester Minou, voici comment faire. C'est simple, et ça marche sur Windows, Mac, et Linux.

Installation rapide :

# 1. Cloner le repo
git clone https://github.com/Tiger-Foxx/minou.git
cd minou

# 2. Installer les dépendances
pip install -r requirements.txt

# 3. Lancer Minou
python main.py

Configuration IA Optionnelle :

  1. Allez sur Google AI Studio
  2. Créez un compte et générez une clé API gratuite
  3. Menu Minou → Paramètres → Collez votre clé
  4. Redémarrez pour activer l'IA complète

Premiers Pas :

  • Double-cliquez sur Minou pour ouvrir le chat
  • Clic droit sur l'icône système pour le menu complet
  • Glissez-déposez Minou pour le déplacer
  • Tapez "Paramètres" dans le chat pour la config
