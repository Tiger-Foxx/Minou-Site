FAQ

Tout sur Minou

Comment couper le son si Minou est trop bruyant ?

Activez le Mode Tranquille depuis le menu pour un silence complet, ou désactivez simplement les sons dans les paramètres.

Comment personnaliser Minou complètement ?

Dans les Paramètres : changez son nom, choisissez chat/chien, réglez vitesse, fréquence des mouvements, sons, et même sa personnalité !

Ça marche sur Linux/macOS ?

Oui ! Clonez le repository GitHub et suivez les instructions d'installation Python. L'installateur Windows est juste plus pratique.

L'IA est obligatoire ?

Non ! Minou fonctionne parfaitement hors-ligne avec des réponses de base. L'IA Gemini est optionnelle pour des conversations avancées.
